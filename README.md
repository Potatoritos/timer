keyboard driven stopwatch timer

available at: https://potatoritos.github.io/timer/

# Controls

`␣`: Start/stop

`r`: Reset

`y`: Copy elapsed time

`e`: Toggle editing mode

In editing mode:

`j` or `←`: Move left

`l` or `→`: Move right

`0-9`: Replace highlighted digit with pressed digit

`h`: Move to hours

`m`: Move to minutes

`s`: Move to seconds

`d`: Move to deciseconds
